# Tutorials
## 这是什么样的`Repository`？
找教程，是一件超级麻烦的事情。

所以我就想做一个教程目录，不同于其他教程，每一个教程我都去亲自学习过。

努力做到通俗易懂，不像很多教程，在讲解知识点的时候，总是引入一些未讲解的知识点。

在程序中，这叫`引用未声明变量`，是要报错的。

总而言之

**这是一个让你省心、通俗易懂的教程集锦。**
